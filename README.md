# Online Courses Platform

Платформа для создания и прохождения онлайн-курсов с системой тестирования и отслеживания прогресса.

Проект создан Шамардиной Елизаветой и Васильевым Владимиром.

## Описание проекта

Online Courses Platform — это серверное приложение, позволяющее пользователям:

- Создавать курсы с модулями и уроками
- Добавлять тестирование к урокам для проверки знаний
- Проходить уроки и отмечать их как завершенные
- Проходить тестирование и отслеживать результаты попыток
- Отслеживать прогресс прохождения курсов
- Просматривать список студентов, проходящих курс

Приложение написано на TypeScript, использует:

- Express.js — для построения REST API
- Mongoose — для удобной работы с базой данных MongoDB
- MongoDB — в качестве основной базы данных
- Docker — для контейнеризации приложения

## Структура проекта

```
src/
├── config/          # Конфигурация
├── models/          # Mongoose модели
├── controllers/     # Бизнес-логика и обработка запросов
├── routes/          # Маршруты API
├── app.ts           # Настройка Express приложения
└── server.ts        # Точка входа сервера
```

## Модели данных

- **User** — пользователи платформы
- **Course** — курсы, созданные пользователями
- **Module** — модули курса
- **Lesson** — уроки в модулях
- **Quiz** — викторины для уроков
- **QuizAttempt** — попытки прохождения викторин
- **UserProgress** — прогресс пользователя по урокам

## Установка и запуск

### Требования
- Node.js (версия 20 или выше)
- MongoDB (локально или через Docker)

### Локальная установка

1. Клонируйте репозиторий и установите зависимости:
```bash
git clone <repository-url>
cd online-courses-project
npm install
```

2. Создайте файл `.env` в корне проекта:
```
DATABASE_URL=mongodb://localhost:27017/online-courses
PORT=3000
```

3. Запустите MongoDB:
```bash
docker-compose up -d mongo
```

4. Запустите приложение:
```bash
npm run dev
```

### Запуск с Docker

```bash
docker-compose up -d
```

Приложение будет доступно на порту 5000.

## Тестирование

Для запуска тестов:
```bash
npm test
```

Импортируйте коллекцию Postman из файла `online-courses-project.postman_collection.json` для удобного тестирования API.

## Технологический стек

- **TypeScript** — типизированный JavaScript
- **Express.js** — веб-фреймворк для Node.js
- **Mongoose** — ODM для MongoDB
- **MongoDB** — NoSQL база данных
- **Jest** — фреймворк для тестирования
- **Supertest** — библиотека для тестирования HTTP запросов
- **Docker** — контейнеризация приложения
- **Morgan** — HTTP logger middleware
- **CORS** — middleware для обработки CORS запросов
