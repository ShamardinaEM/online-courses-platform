# Online Courses Platform

Платформа для создания и прохождения онлайн-курсов с системой тестирования и отслеживания прогресса.

Проект создан Шамардиной Елизаветой и Васильевым Владимиром.

## Описание проекта

Online Courses Platform — это серверное приложение, позволяющее пользователям:

- Создавать курсы с модулями и уроками
- Добавлять тестирование к урокам для проверки знаний
- Проходить уроки и отмечать их как завершенные
- Проходить тестирование и отслеживать результаты попыток
- Отслеживать прогресс прохождения курсов
- Просматривать список студентов, проходящих курс

Приложение написано на TypeScript, использует:
- Typescript + Zod
- MongoDB — в качестве основной базы данных
- Docker — для контейнеризации приложения


## Модели данных

- **User** — пользователи платформы
- **Course** — курсы, созданные пользователями
- **Module** — модули курса
- **Lesson** — уроки в модулях
- **Quiz** — викторины для уроков
- **QuizAttempt** — попытки прохождения викторин
- **UserProgress** — прогресс пользователя по урокам

## Установка и запуск

### Требования
- Node.js (версия 20 или выше)
- MongoDB (локально или через Docker)

### Локальная установка

1. Клонируйте репозиторий и установите зависимости:
```bash
git clone <repository-url>
cd online-courses-project
npm install
```

2. Создайте файл `.env` в корне проекта:
```
MONGO_URL=mongodb://localhost:27017/
```

3. Запустите MongoDB:
```bash
docker-compose up -d
```

4. Запустите тесты:
```bash
npm run test
```
